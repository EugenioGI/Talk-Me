Pagina dedicada a personalizar pictogramas de ayuda para personas con distintas discapacidades.

El usuario registra sus pictogramas favoritos, crea nuevos de acuerdo a sus necesidades y tiene posibilidad de actualizarlos.

El rol del administrador puede gestionar a sus usuarios y ver pictogramas mas usados y administrar las discapacidades de acuerdo a los pictogramas mas usados.
